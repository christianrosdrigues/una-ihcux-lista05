# Global Exchange - Conversor

## Heurísticas de UX Aplicadas

### 1. Visibilidade do Status do Sistema
O sistema informa ao usuário o que está acontecendo, exibindo mensagens como:
- "Conectando ao Banco Central..."
- "Calculando taxas..."

### 2. Prevenção de Erros
O uso do try-catch evita que o programa quebre quando o usuário digita valores inválidos, exibindo uma mensagem amigável.

### 3. Estética e Design Minimalista
O sistema utiliza cores e organização visual simples para destacar informações importantes, como o resultado final.
